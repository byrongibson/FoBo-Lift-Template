About This Demo

This demo demonstrates and uses a Lift easily integratable external/optional module called FoBo that quick and easy will integrate some great front-end toolkit(s) into your Lift application, giving you the flexibility and choice of setting up and use the toolkit (and version) of your liking. For more information about the FoBo module take a look at the readme at Github.
At this writing 2012-02-05 this demo is just a very basic demo showing a few of the most obvious and commonly used features (actually only two of them) found in the included toolkits.

FoBo and the fobo mix

This page uses the fobo.css from the fobo module, it also make heavy use of a application specific css styling using the fobo-demo-app1.css file (external to the module).
The fobo.css setup option in the FoBo module is a love, peace and harmony mix of the ZURB/Foundation and the Twitter Bootstrap front-end toolkit.

The ZURB/Foundation toolkit

The ZURB/Foundation toolkit is a great mobile scalable toolkit. Apart from giving you easy access and integration of the fobo mix the FoBo module also gives you the setup option to use the foundation toolkit separately.

The Twitter Bootstrap toolkit

The Twitter Bootstrap toolkit is a great toolkit with lots candy. Apart from giving you easy access and integration of the fobo mix the FoBo module also gives you the setup option to use the foundation toolkit separately.

Scroll

Scroll the page to see the topbar scrollspy function provided by bootstrap. The scroll function may fall back to a static topbar if you are on a smart phone or tablet.

Drag

Drag the window to see foundations mobile scalabilty and reactive fluid grid function. If you have not done it allredy open this page in your favorite smart phone.
