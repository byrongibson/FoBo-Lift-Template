About This Demo

This demo demonstrates a optional Lift, generic front-end toolkit module FoBo that quick and easy will integrate some great front-end toolkit(s) into your Lift application, giving you the flexibility and choice of setting up and use the toolkit (and version) of your liking. For more information about the FoBo module take a look at the readme at Github.
Check out the integrated toolkit documentation for a comprehensive list of the, in this demo included, toolkits features.

FoBo and the fobo mix

This page uses the FoBo02 ToolKit init param. option of the fobo module which make use of the fobo.css, there is also separate BootstrapXX, FoundationXX options and more. This demo also make heavy use of a application specific css styling using the fobo02-demo-app1.css file (external to the module).
The fobo.css setup option in the FoBo module is a love, peace and harmony mix of the ZURB/Foundation and the Twitter Bootstrap front-end toolkit.

To initiate the module chose the toolkit option you like to use and call FoBo.init(). This demo uses the following module setup in lift Boot:


                           import net.liftmodules.FoBo._

                           /**
                            * A class that's instantiated early and run.  It allows the application
                            * to modify lift's environment
                            **/
                            class Boot extends Loggable {
                              def boot {

                                //If using defaults FoBo init params can be omitted
                                FoBoInitParams.JQuery=JQuery171  
                                FoBoInitParams.ToolKit=FoBo020 
                                FoBoInitParams.ToolKit=PrettifyJun2011
                                FoBoInitParams.ToolKit=JQueryMobile101
                                FoBo.init()
                                  :

In Lift's html5 template:s add (depending on your setup) something like this:

                        <!DOCTYPE html>
                        <html>
                          <head>
                             :
                             <!--Main css for the FoBo020 module option-->
                             <link href="/classpath/fobo/fobo.css" type="text/css" rel="stylesheet">
                             <!--app specific tweeks and overrides external to module goes here-->
                             <link href="[path to your projects app css]" type="text/css" rel="stylesheet">  
                             :                        
                          </head>
                          <!--Demo page setup for prettyprint and bootstrap scrollspy-->
                          <body data-spy="scroll" data-target="#fobo-navbar" data-offset="40" onload="prettyPrint()">
                             :
                             <!--In end of page (for fast pageload) add the scripts you need--> 
                             <script src="/classpath/fobo/prettify.js" type="text/javascript"></script>
                             <script src="/classpath/fobo/lang-scala.js" type="text/javascript"></script>
                             <script src="/classpath/fobo/bootstrap-scrollspy.js" type="text/javascript"></script>
                             :
                          </body>        
                        </html>
For a more information see the FoBo modules Github.

The ZURB/Foundation toolkit

The ZURB/Foundation toolkit is a great mobile scalable toolkit. Apart from giving you easy access and integration of the fobo mix the FoBo module also gives you the setup option to use the foundation toolkit separately take a look at the Foundation demo .

Foundation

The Twitter Bootstrap toolkit

The Twitter Bootstrap toolkit is a great toolkit with lots candy. Apart from giving you easy access and integration of the fobo mix the FoBo module also gives you the setup option to use the bootstrap toolkit separately. The included bootstrap documentation pages is modified to use the FoBo option Bootstrap demo .

Bootstrap

The JQuery Mobile toolkit

The JQuery Mobile toolkit is a Touch-Optimized Web Framework for Smartphones & Tablets.
Apart from a ToolKit and JQuery module init option for you to chose your favorite toolkit and jquery version from there is also a additional MobileToolKit init option in the fobo module for you to chose a mobile centric toolkit from. To demonstrate the usage the (in this demo) included JQuery Mobile documentation pages is modified to use the Fobo modules MobileToolKit option JQuery-mobile demo .
