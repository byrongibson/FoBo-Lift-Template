About This Demo and the FoBo module

This pages demonstrates a optional Lift, generic front-end toolkit module FoBo that quick and easy will integrate some of the industry leading web-centric open source front-end toolkits into your Lift application, giving you the flexibility and choice of setting up and use the toolkit:s (and version:s) of your liking, that is, if you like you can even setup and potentially use several toolkits simultaneously. Added to this is a growing number of FoBoAPI functions (optional to use) that will gives you even more ease of usage, power and control. For some examples and setup information see below.

Except from the Setup section below and the FoBoAPI the most up to date way to explore and learn how to use the FoBo module would be to fork or study this demo code on Github. There is also a cleaner FoBo Bootstrap only basic Lift Template (for quick start lift lift project with bootstrap ) available here.

For more information about the FoBo module take a look at the FoBoAPI and the readme on Github.

Also check out the integrated toolkit documentation for a comprehensive list of the, in this demo included, toolkits features.

Mobile Scalability with FoBo

The FoBo mix - One of the convenience options the FoBo Lift module comes with is the FoBo02 setup option. The FoBo02 setup option is a love, peace and harmony mix of the ZURB/Foundation and the Twitter Bootstrap front-end toolkit. This concert mix will give you the awesomeness of foundation with its rock & roll solid reactive, mobile scalability and the brilliance of bootstrap, it's components and interactions.

This demo uses (among others) the FoBo02 Toolkit option of the FoBo module but that is just one of several Toolkit setup options, there is also separate BootstrapXX, FoundationXX, JQueryMobileXXX Toolkit options and more. This demo also make use of a application specific css styling file (external to the module).

The ZURB/Foundation toolkit

The ZURB/Foundation toolkit is a great reactive and mobile scalable toolkit.

The FoBo module also gives you the (separate) setup option FoundationXXX for using the foundation toolkit. For a demonstration take a look at the included Foundation demo .

Foundation

The Twitter Bootstrap toolkit

The Twitter Bootstrap toolkit is a great toolkit with lots of candy. It's simple and flexible and have popular user interface components and interactions.

Apart from giving you the easy access and integration of the FoBoXX mix the FoBo module also gives you the BootstrapXXX setup option for using the bootstrap toolkit. The bootstrap documentation pages in this demo is slightly modified to use the FoBoXX mix option but can as easily (or even more so) be set up and used with the BootstrapXXX option. For a example see Templating-With-Twitter-Bootstrap project it uses the FoBo module with the Bootstrap200 setup option.

FoBo:s build system has hooks into LESS to dynamicaly generate the bootstrap.css so if you like you can make some serious tweaks to the generated bootstrap.css. See bootstrap demo and documentation for more info on how to use Less with bootstrap.

Bootstrap

The JQuery Mobile toolkit

The JQuery Mobile toolkit is a Touch-Optimized Web Framework for Smartphones & Tablets.

To use this mobile centric toolkit set the JQueryMobileXXX option and go mobile.

The JQuery-mobile demo pages in this demo is modified to use the JQueryMobile101 option.

JQuery-mobile

Google Code Prettify

Google Code Prettify A Javascript module and CSS file that allows syntax highlighting of source code snippets in an html page. You will find it used in many places in this demo.

DataTables

DataTables is a plug-in for the jQuery Javascript library. It is a highly flexible tool, based upon the foundations of progressive enhancement, which will add advanced interaction controls to any HTML table.

The DataTables demo pages in this demo is modified to use the DataTables190 option.

DataTables

Knockout JS

Knockout - Simplify dynamic JavaScript UIs by applying the Model-View-View Model (MVVM) pattern. Some of Knockouts key concepts is:

  • Declarative Bindings - Easily associate DOM elements with model data using a concise, readable syntax.
  • Automatic UI Refresh - When your data model's state changes, your UI updates automatically.
  • Dependency Tracking - Implicitly set up chains of relationships between model data, to transform and combine it.
  • Templating - Quickly generate sophisticated, nested UIs as a function of your model data.
